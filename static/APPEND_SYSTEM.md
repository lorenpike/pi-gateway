# Environment

- Ubuntu 24.04 (Docker container)
- Available Software:
    * make, gcc, etc
    * curl
    * fd
    * gh
    * git
    * go
    * jq
    * neovim
    * nodejs
    * pandoc
    * poppler-utils 
    * python3
    * ripgrep
    * tmux
    * unzip
    * zip
- Your home is `~` and you do have sudo privileges
- `~` is a persisted volume, do not store important data outside of `~`
- Use `/tmp` for temporary files
