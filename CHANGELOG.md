## [0.1.0] - 2026-07-15

### Added

- Initial wall-e gateway with HTTP, Telegram, and Discord channels.
- Persistent pi sessions, media handling, scheduled jobs, and supervised
  container services.
- Versioned Docker, documentation, and GitHub release workflow.
- Portable deployment, home-volume backup, restore, and computer-migration
  tooling.
